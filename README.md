# SnapList

A simple, clean, and mobile-friendly productivity app that helps you manage your daily tasks and shopping lists in one place.

SnapList is designed with a minimal interface, making it easy to organize your day, keep track of groceries, and stay productive without unnecessary complexity.

---

# Features

* Create and manage daily tasks
* Maintain a separate shopping list
* Mark items as completed
* Edit items directly without pop-up dialogs
* Delete individual items
* Search through tasks and shopping items
* Automatic date and time for every entry
* Light and Dark mode
* Local storage support (data is saved in your browser)
* Responsive design for desktop and mobile devices

---

Simply open `index.html` in your browser.

No installation or additional dependencies are required.

---

# How to Use

1. Open the application.
2. Choose between the **Tasks** or **Shopping** tab.
3. Add a new item.
4. Tap the checkbox to mark it as completed.
5. Tap the item text to edit it.
6. Use the search button to quickly find items.
7. Switch between Light and Dark mode anytime.
8. Delete items individually or clear the entire list.
