# SnapList Quick Start Guide

## Getting Started

### Running the Application
1. **Option 1: Direct Browser Access**
   - Open `index.html` directly in your browser

2. **Option 2: Local Server (Recommended)**
   ```bash
   python -m http.server 8000
   ```
   - Then navigate to `http://localhost:8000` in your browser

## Basic Usage

### Adding Tasks
1. Select the appropriate tab (📝 To-Do or 🛒 Cart)
2. Enter task text in the input field
3. Click "Add" or press Enter

### Completing Tasks
- Click the checkbox next to any task to mark it as complete
- Click again to mark as incomplete

### Editing Tasks
- Click directly on any task text to edit it
- Changes are saved automatically

### Deleting Tasks
- Hover over any task
- Click the red delete button that appears

## Enhanced Features

### Setting Priority
1. Select priority level (High/Medium/Low) from the dropdown
2. Add your task
3. Visual indicators help identify priority levels

### Setting Due Dates
1. Use the date picker to select a due date
2. Add your task
3. Overdue items are highlighted in red

### Using Categories
1. Click "Add Category" in the Categories section
2. Enter a name and select a color
3. When adding tasks, select a category from the dropdown

### Adding Tags
1. Enter comma-separated tags in the tags input field
2. Click "Add Tags" when adding a task
3. Use tag filtering to find related tasks

### Breaking Tasks into Steps
1. Add a task normally
2. Use the "Add a step..." input below any task
3. Check off steps as you complete them
4. Progress is shown as completed/total steps

## Organization & Navigation

### Switching Between Lists
- Click the 📝 To-Do tab for general tasks
- Click the 🛒 Cart tab for shopping items

### Sorting Tasks
1. Choose a sorting option from the dropdown:
   - Creation Date
   - Due Date
   - Priority
   - Status
   - Alphabetical
2. Toggle between ascending (↑) and descending (↓) order
3. Click "Reset" to return to default sorting

### Searching Tasks
1. Click the 🔍 search button
2. Enter search text
3. Results update automatically

### Filtering Tasks
1. Select a category from the filter dropdown
2. Enter tags in the tag filter input
3. Click "Clear Filters" to reset

## Backup & Restore

### Exporting Data
1. Scroll to the "Backup & Restore" section
2. Click "Export as JSON" for complete backup
3. Click "Export as CSV" for simple data transfer

### Importing Data
1. Click "Import Data"
2. Select a previously exported file
3. Your data will be restored automatically

## Customization

### Theme Switching
- Toggle the switch in the top right to switch between:
  - Light theme (default)
  - Dark theme

## Tips & Tricks

### Keyboard Shortcuts
- Press Enter in input fields to quickly add items
- Click directly on text to edit without additional buttons

### Data Persistence
- All data is automatically saved in your browser
- Data persists between browser sessions
- Clearing browser data will remove your tasks

### Mobile Usage
- SnapList is fully responsive
- Works on phones, tablets, and desktops
- Touch-friendly interface

## Troubleshooting

### Common Issues
- **Tasks not saving**: Ensure browser storage is enabled
- **Features not working**: Try refreshing the page
- **Import fails**: Check file format and integrity

### Browser Compatibility
SnapList works in all modern browsers:
- Chrome 50+
- Firefox 50+
- Safari 10+
- Edge 15+

### Data Security
- All data is stored locally in your browser
- No data is sent to external servers
- Exported files should be stored securely
